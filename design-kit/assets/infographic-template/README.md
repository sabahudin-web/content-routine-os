# Infographic Templates

**Last updated:** 2026-04-12

Template folder for LinkedIn and newsletter infographic visual templates.

## Status

No templates yet. Add infographic PDF/image examples here as they're created.

## Design System

All infographics follow `design-kit/design-tokens.md` for colors, fonts, and spacing. Key specs:
- LinkedIn format: 1080px wide, 4:5 portrait (1080x1350)
- Newsletter format: 16:9 landscape
- Alfa Slab One for headlines, Schoolbell for body
- Grain texture on every background
- Mandatory footer with profile photo + Unstuck logo
