## The 6 Winning Humor Patterns

### Pattern 1 — Absurd Math / The "Struggling" Format
Take something real (an expense, a time allocation, a behavior) and frame it as absurdly high while everything "normal" is tiny.

**Structure:**
```
[Normal thing] = [tiny amount]
[Normal thing] = [tiny amount]
[The absurd thing] = [outrageously high amount]

How is anyone supposed to live like this?
```

---

### Pattern 2 — Self-Deprecating Confession
Admit something embarrassing but universally true. Set up the expectation. Subvert it with the absurd choice.

**Structure:**
```
[Expected occasion / moment]

[What normal people would do]

Nah. I did the [absurd tech/work choice].

[The actual punchline]

[Self-aware outro — "Love is temporary. [X] is forever." style]
```

---

### Pattern 3 — Dialogue Format
Write a short scene between two people. The humor is in one person calling out the other.

**Structure:**
```
[Person A]:
"[Setup line — often showing LinkedIn brain or work obsession]"

[Person B]:
"Stop talking to me like you're on [platform]."

[Person A]:
"..."

[Person B]:
"[Short, grounding reality check]"

[Person A]:
"Yes."
```

---

### Pattern 4 — One-Liner Parody
Parody a common LinkedIn cliché with one sentence.

**Structure:**
```
[Type of person] be like:

"[Exaggerated version of what they actually say — sounds almost real but slightly too much]"
```

---

### Pattern 5 — Insider Test / Gatekeeping
Create an in-group/out-group moment. If you get the joke, you're one of us.

**Structure:**
```
How to spot a fake [expert]:

[Simple test that only real practitioners would know or find funny]

[Playful roast if they don't get it]
```

---

### Pattern 6 — Third-Person Reveal
Build tension about someone who did something dramatic. Then reveal it was you.

**Structure:**
```
Today I'm going to tell you the story of [the person who did X].

[Dramatic details that make them sound like someone else]

Well... [that person] is me.

[Quick self-aware lesson — usually about accepting failure]
```

---

## Humor Modes

**Mode A — Observation-based:** You notice something absurd about how people (including you) behave. Short. No lesson. Just the observation.

**Mode B — Story-based:** A real thing happened. It was embarrassing or dumb. You're telling it now. The humor is in the honesty.
