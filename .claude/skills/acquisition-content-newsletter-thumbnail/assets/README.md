# Newsletter Thumbnail Assets

Add your logo and example thumbnails here.

**logo.png** — Your logo file (used for brand reference in thumbnail generation)
- Recommended: PNG, at least 500x500px
- Add as: `logo.png`

**thumbnail-example.png** — A reference thumbnail showing your style
- Optional but helpful — helps Nano Banana match your visual style
- Add as: `thumbnail-example.png`

Run `/onboard` — during the Design System phase, Claude will guide you to add your files here.
