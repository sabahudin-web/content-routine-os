# YouTube Video Description Writer

Write descriptions that rank in search and convert viewers into subscribers, customers, and community members.

## Required Information

Before writing, gather:

| Required | Why |
|----------|-----|
| Video topic/title | For SEO keyword integration |
| Target keywords | For search ranking |
| Primary CTA | What action should viewers take? |
| Links to include | Products, affiliates, resources mentioned |

| Optional | Improves |
|----------|----------|
| Channel name/links | Branding consistency |
| Social media links | Cross-platform growth |
| Timestamps | Watch time and navigation |
| Affiliate disclosure needs | Legal compliance |

## Description Structure (13 Parts)

### Above the Fold (Parts 1-2)

Viewers see these WITHOUT clicking "Show more". Make them count.

**Part 1: SEO Hook**
First line must: (1) describe what's in the video, (2) include searchable keywords.

```
Bad: "In this video I talk about some cool stuff"
Good: "How to start a YouTube channel in 2024 - complete beginner guide with equipment list and upload checklist"
```

**Part 2: Primary CTA**
Second line drives action. One clear call to action with link.

### Resource Section (Part 3)

**Part 3: Products/Links Mentioned**
List everything referenced in the video. Use clear headers.

```
=============================
Products and Gear in This Video
=============================
[Category]
- [Product] → [Link]
```

### Channel Growth (Parts 4-6)

**Part 4: Subscribe CTA**
**Part 5: Ways to Work Together** — Free / Paid / High-ticket tiers
**Part 6: Social Links**

### Engagement (Part 7)

**Part 7: Comment Question**
Drive comments with a specific, easy-to-answer question.

### Monetization (Parts 8-9)

**Part 8: Featured Affiliate** — Highlight one high-value affiliate
**Part 9: Short Disclaimer** — Required for FTC compliance

### Navigation (Parts 10-11)

**Part 10: Timestamps**
CRITICAL: Must start with 0:00 or timestamps won't work.

**Part 11: Hashtags**
3-5 relevant hashtags. Mix broad and specific.

### Channel Info (Parts 12-13)

**Part 12: About Section** — Brief channel description with SEO keywords
**Part 13: Full Disclaimer** — Detailed legal disclaimer

**Save output to:** `inbox/outputs/md/YYYY-MM-DD-acquisition-content-youtube-description.md`

## Writing Rules

1. **Keywords first** - Front-load SEO terms in Part 1
2. **One CTA above fold** - Don't dilute with multiple asks
3. **Timestamps start at 0:00** - Non-negotiable for YouTube chapters
4. **Affiliate disclosure required** - Both short (Part 9) and full (Part 13)
5. **Use visual separators** - `=====` headers improve scannability
6. **No more than 5 hashtags** - Looks spammy beyond that
7. **Question must be specific** - "What do you think?" gets ignored; specific questions get answers

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Generic first line | Include specific keywords people search |
| Missing 0:00 timestamp | Chapters won't display without it |
| No affiliate disclosure | Legal risk - always include |
| Too many CTAs above fold | Pick ONE primary action |
| Hashtags in body text | Place at end only |
| No engagement question | Miss comment algorithm boost |
