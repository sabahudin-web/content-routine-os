# Naming Convention Rule

## Skills
Every skill name starts with its area prefix:
- offer-* (e.g., offer-create-proposal)
- acquisition-* (e.g., acquisition-score-enrich)
- conversion-* (e.g., conversion-call-prep)
- delivery-* (e.g., delivery-onboard)
- operations-* (e.g., operations-morning-cafe)
- finance-* (e.g., finance-track-revenue)

## Files
- Kebab-case only: proposal-acme.md, not Proposal Acme.md
- Date prefix for outputs: 2026-03-24-filename.md
- No spaces, no CamelCase, no underscores

## Folders
- All lowercase, kebab-case
- Descriptive: onboarding/, not on/, not ob/
